# Project Setup & Development Guide

## Project Overview

This repository contains a modern web application built with a fast and scalable frontend stack. The project is optimized for clean UI development, type safety, and smooth deployment.

---

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

---

## Getting Started

### Prerequisites

Ensure you have the following installed:

- Node.js (v18 or higher recommended)
- npm

You can install Node.js using nvm:  
https://github.com/nvm-sh/nvm#installing-and-updating
